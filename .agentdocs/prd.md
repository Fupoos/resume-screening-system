# 简历智能初筛系统 - 产品需求文档 (PRD)

## 一、产品概述

### 1.1 产品背景

企业HR在招聘过程中面临以下痛点：
- **邮箱简历管理混乱**：大量简历散落在邮箱中，难以统一管理
- **初筛效率低下**：人工逐一查看简历耗时耗力
- **评估标准不一致**：不同HR对简历的判断标准存在差异
- **优秀候选人遗漏**：人工筛选容易错过符合条件的候选人

### 1.2 产品目标

构建一个自动化的简历初筛系统，通过监听企业邮箱自动获取简历，调用AI Agent进行智能评估，帮助HR快速识别优秀候选人。

### 1.3 目标用户

- **主要用户**：企业HR、招聘负责人
- **次要用户**：部门负责人（查看筛选结果）

## 二、核心功能需求

### 2.1 邮箱监听与简历获取

**需求描述**：系统自动监听企业邮箱，获取简历附件并解析。

**功能点**：
- 支持腾讯企业邮箱（IMAP协议）
- 每5分钟自动检查新邮件
- 自动下载PDF/DOCX格式的简历附件
- 已处理邮件自动移动至"已处理"文件夹
- **核心原则**：只保留有PDF附件+正文内容的简历

**验收标准**：
- 能正确连接企业邮箱
- 能识别并下载简历附件
- 能处理多种简历格式（PDF、DOCX）
- 系统稳定运行，无遗漏

### 2.2 简历解析与信息提取

**需求描述**：自动解析简历文件，提取候选人关键信息。

**提取信息**：
- 基本信息：姓名、性别、电话、邮箱
- 教育背景：学校、学历、专业、毕业时间
- 工作经历：公司、职位、工作时间、工作职责
- 项目经历：项目名称、角色、项目时间、项目描述
- 技能标签：技术技能、熟练度

**验收标准**：
- 姓名提取准确率 ≥ 95%
- 电话、邮箱提取准确率 = 100%
- 学历、工作经历提取准确率 ≥ 90%
- 支持常见简历格式

### 2.3 AI Agent 智能评估

**需求描述**：调用外部AI Agent对简历进行评分和筛选建议。

**评估维度**：
- 技能匹配度
- 工作经验相关性
- 教育背景匹配度
- 综合素质评价

**评分规则**：
- 90-100分：完全匹配，可以进入面试
- 70-89分：基本匹配，可以考虑面试
- 50-69分：部分匹配，需要进一步评估
- 0-49分：不匹配，不建议面试

**筛选结果**：
- **PASS**：Agent评分 ≥ 70
- **REVIEW**：40 ≤ Agent评分 < 70
- **REJECT**：Agent评分 < 40

**核心原则**：所有评分必须通过外部Agent完成，系统不进行本地评分。

**验收标准**：
- Agent调用成功率 ≥ 95%
- 评分结果实时返回
- 支持重试机制（最多3次）
- Agent不可用时能够降级处理

### 2.4 筛选结果展示

**需求描述**：HR可以查看所有简历的筛选结果。

**功能点**：
- 列表展示：候选人姓名、职位、评分、筛选结果
- 筛选功能：按筛选结果、职位、城市筛选
- 详情查看：查看候选人完整信息和评估建议
- 技能标签展示：显示候选人核心技能（最多3个）

**验收标准**：
- 数据实时更新
- 筛选功能正常工作
- 详情信息完整展示

### 2.5 仪表盘统计

**需求描述**：提供数据统计分析功能。

**统计维度**：
- 概览统计：总简历数、通过数、待定数、拒绝数、通过率
- 按城市统计：各城市的简历数量和质量分布
- 按职位统计：各职位的简历数量和质量分布

**验收标准**：
- 数据准确无误
- 统计实时更新
- 可视化展示清晰

### 2.6 岗位管理

**需求描述**：HR可以管理招聘岗位信息。

**功能点**：
- 创建/编辑/删除岗位
- 设置岗位要求（技能、学历、经验）
- 配置Agent评估参数

**验收标准**：
- CRUD操作正常
- 岗位信息正确保存
- Agent配置生效

### 2.7 人工审核

**需求描述**：处理无法自动解析的简历（加密PDF、扫描件、图片格式简历）。

**功能点**：
- 显示所有需要人工审核的简历（`raw_text <= 100字符`）
- 查看原始附件文件
- 手动编辑候选人信息：姓名、电话、邮箱、学历、工作年限、技能标签
- 删除无效记录

**触发场景**：
- DOCX文件是图片格式（扫描件）
- PDF文件加密无法解析
- 其他解析失败的情况

**验收标准**：
- 能正确识别需要人工审核的简历
- 编辑后信息正确保存
- 支持删除无效记录

## 三、非功能性需求

### 3.1 性能要求

- 简历解析时间：< 30秒/份
- API响应时间：< 500ms（P95）
- 系统可用性：≥ 99%

### 3.2 安全要求

- 邮箱授权码AES-256加密存储
- 数据传输HTTPS加密
- 敏感信息脱敏展示

### 3.3 兼容性要求

- 支持主流浏览器（Chrome、Firefox、Safari、Edge）
- 支持移动端查看

## 四、功能优先级

### P0（必须有）
- 邮箱监听与简历获取
- 简历解析与信息提取
- AI Agent评估
- 筛选结果展示

### P1（重要）
- 仪表盘统计
- 岗位管理
- 人工审核

### P2（可选）
- 用户认证与权限管理
- 企业微信/邮件通知
- 导出功能

## 五、成功指标

- **效率提升**：简历初筛时间缩短 80%
- **准确性**：Agent评估与人工评估一致性 ≥ 85%
- **用户体验**：系统易用性评分 ≥ 4.0/5.0
